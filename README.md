# FMCTO-Dataset
The FMCTO-Dataset is a supervised learning dataset designed for research on reliable collaborative offloading in edge computing. Constructed from the publicly available VEC Edge Server Offloading Dataset, it transforms raw offloading records—including latency, energy consumption, target edge server details, and mobility status—into four normalized criterion scores. It further generates a continuous label known as the Task Completion Reliability Index (TCRI).
This dataset supports research tasks such as fuzzy measure learning, Choquet integral modeling, multi-criteria decision analysis, task reliability prediction, and edge server resource availability assessment.
The public version contains 2,856 valid offloading task samples; each sample comprises a task ID, four criterion scores, and a composite reliability label. The four criteria are: x1_delay_score, x2_energy_score, x3_resource_availability_score, and x4_link_stability_score. All criteria are positive indicators ranging from 0 to 1, where higher values ​​signify conditions more favorable to reliable offloading. The label y_tcri represents the Task Completion Reliability Index—also ranging from 0 to 1—reflecting the reliability of task completion under soft deadline constraints.
A key feature of this dataset is the transformation of raw physical measurements into multi-criteria reliability data suitable for direct use in supervised learning. Unlike datasets containing only raw offloading records, this dataset provides feature vectors and target labels specifically designed for reliability modeling; these can be directly used to train and evaluate fuzzy measures, Choquet integrals, regression models, and other multi-criteria aggregation models. The resource availability score is calculated using only historical records from the specific edge server prior to the current task's arrival, thereby avoiding the use of future information. The TCRI label is generated via a deterministic mathematical mapping rather than subjective expert scoring, ensuring good reproducibility.
It should be noted that this is a derived dataset based on the original VEC Edge Server Offloading Dataset, rather than newly collected real-world vehicular edge computing data. When publishing or using this dataset, the source of the original data must be acknowledged, and the original dataset's license agreement and citation requirements must be observed.
